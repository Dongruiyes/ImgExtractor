# 🖼️ ImgExtractor - DLL 资源提取工具

**一款轻量级的 DLL 文件资源提取工具，支持多种图片格式的批量提取**

## ✨ 核心功能

### 📂 单文件提取
- 支持从单个 `.dll` 文件中提取嵌入式图片资源
- 可导出格式：`.png`、`.ico`、`.jpg` 等常见图片格式
- 支持自定义输出目录

### 🔍 批量处理
- 递归遍历指定文件夹下的所有 `.dll` 文件
- 自动识别并提取所有嵌入的图片资源

### 🗃️ 资源管理
- 支持对指定目录的图片文件进行批量复制
- 可筛选特定扩展名的图片文件（`.png`/`.ico`/`.jpg` 等）
- 提供统一的资源整理输出目录
